# Проект "Растровые алгоритмы для построения линий и окружностей"

## Описание
Приложение,иллюстрирующее работу базовых алгоритмов растеризации отрезков и кривых.

## Цель работы
Разработать приложение/веб-приложение для иллюстрации работы базовых растровых алгоритмов: пошаговый алгоритм, алгоритм ЦДА, алгоритм Брезенхема, алгоритм Бразенхема для окружностей.

## Задачи работы
- Реализовать пошаговый алгоритм для рисования отрезков.
- Реализовать алгоритм ЦДА для рисования отрезков.
- Разработать алгоритм Брезенхема для построения отрезков.
- Разработать алгоритм Брезенхема для построения окружностей.
- Создать пользовательский интерфейс с функциями масштабирования, вывода системы координат, осей, линий сетки и их подписей.

## Использованные средства разработки
- Java
- javax.swing
- java.awt

## Ход работы
- Реализованы алгоритмы для построения отрезков.
- Создан пользовательский интерфейс с функциями масштабирования и отображения системы координат и сетки.
- Подготовлен краткий отчет с временными характеристиками работы алгоритмов.
- Проведено ручное тестирование.

## Вывод
В ходе работы были приобретены практические навыки по реализации растровых алгоритмов, разработке пользовательского интерфейса и подготовке отчетов. Углублены знания в области компьютерной графики.

### Измерения времени выполнения
- Среднее время выполнения для алгоритма пошагового метода: 420ms
- Среднее время выполнения для алгоритма ЦДА: 430ms
- Среднее время выполнения для алгоритма Брезенхема: 450ms
- Среднее время выполнения для алгоритма построения окружностей по алгоритму Брезенхема: 450ms
